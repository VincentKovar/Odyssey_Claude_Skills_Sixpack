# Strategic Multiplier Skill

## What This Skill Does

Takes an existing 8-field concept brief and multiplies it into new variants by
swapping the strategic core â€” motivator, benefit, angle, and persona â€” while
keeping the format locked. The core belief: do the deep thinking once, then
multiply. ~60% of a concept's DNA can stay the same across angle variations.

**The formula:**
`1 format Ã— N angles Ã— N personas Ã— N funnel stages = volume without creative decay`

---

## What Gets Locked vs. Swapped

**LOCKED (do not change):**
- Format
- Visual structure
- Hook mechanism
- Setting

**SWAPPED (must change):**
- Concept Name
- Concept Summary
- Motivator
- Benefit
- Angle
- Why It Will Work
- Rationale

## How to Run a Strategic Multiplier

### Step 1 â€” Read the source brief
Identify the Format. This does not change. The visual structure, hook mechanism,
and setting all stay.

### Step 2 â€” Suggest new motivators (then let the user override)
Pull from the client's persona file if available in project context. Suggest
3â€“5 candidate motivators before writing any briefs. Use this format:

> Here are 3 strategic angles I'd suggest based on [client] personas. Let me
> know if you want one of these or if you have a specific motivator in mind:
> 1. [Persona name] â€” "[First-person motivator statement]"
> 2. [Persona name] â€” "[First-person motivator statement]"
> 3. [Persona name] â€” "[First-person motivator statement]"

Wait for confirmation or override before writing the full brief.

### Step 3 â€” Check the funnel stage
Ask or infer whether this variant is for the same stage (TOF/MOF/BOF) or a
different one. If different, the Motivator and Benefit language must shift:

- **TOF (Problem Unaware / Problem Aware):** Lead with the problem. Do not
  mention the product early.
- **MOF (Solution/Product Aware):** Lead with differentiation. Proof matters.
- **BOF (Most Aware):** Lead with offer, urgency, or social proof.

### Step 4 â€” Write the multiplied brief
Output a complete 8-field brief. The Rationale must explicitly name:
- What was locked and why
- What was swapped and why
- Why the new angle works for the new persona/funnel stage

## The Andromeda Rule

Before outputting any multiplied brief, check: does this new variant have the
same Format + Angle + Funnel Stage as any other brief already in context?

If yes: "This combination already exists as [Concept Name]. To avoid creative
decay, I'd suggest varying [dimension] instead. Here are alternatives: ..."

Never output two briefs that would look like the same ad in different clothes.

## Quality Rules

1. **Never output a variant where only the Concept Name changed.** Every
   swapped field must reflect the actual strategic shift.
2. **Rationale bullets must be specific.** "This works because it's relatable"
   is not acceptable. Name the mechanism â€” the persona insight, the creative
   structure, the funnel logic.
3. **The Motivator must sound like something a real person thinks**, not
   something a strategist writes. Test it: would this person actually say this
   to themselves?
4. **Why It Will Work must cite a real creative or behavioral insight** â€” a
   validated ad pattern, a specific psychological mechanism, or a proven
   platform behavior.
5. **When multiplying for a different funnel stage**, the language register must
   shift. TOF ads lead with the world before the product exists. BOF ads lead
   with proof and urgency. Never use the same framing at both ends of the funnel.
6. **The Andromeda Rule applies always.** Diversity across angle and awareness
   level is non-negotiable.
